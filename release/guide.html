📖 소개

이 파워스킬 패키지는 obra/superpowers 원본에서 CLI · 터미널 · 복잡한 설치 과정이 필요한 부분을 모두 제거하고, Claude · GPT · Gemini 공식 홈페이지 또는 API에서 바로 사용할 수 있도록 개조한 수정본입니다.

프로그래밍 경험이 전혀 없는 초심자도 AI와 함께 바이브 코딩을 바로 시작할 수 있으며, RISUAI의 모듈, 플러그인, 봇 카드 등 다양한 콘텐츠 제작에 활용할 수 있습니다.

💡 핵심 전제
배포 파일 중 risuai-manual.skill은 RISUAI 실전 매뉴얼입니다. RISUAI 관련 작업을 할 때는 이 스킬을 반드시 함께 업로드한 상태에서 파워스킬을 사용하세요. AI가 RISUAI의 봇 카드 구조, 모듈, 플러그인 체계를 정확히 이해한 채로 작업합니다.

💬 사용 예시

프롬프트 복사

프롬프트 복사

risuai-manual을 참고해서 모듈 구조에 맞게 설계해줘.

예시 3 — 플러그인 개발 (TDD 방식)

프롬프트 복사

플러그인 구조 설명부터 시작하고, TDD 방식으로 개발 계획을 세워줘.
💡 팁 — 대화 시작 전에 AI에게 다음 문장을 먼저 보내면 좋습니다:
"업로드한 스킬 파일들을 읽고, 관련 스킬이 있으면 반드시 따라서 진행해줘."

🎯 초심자를 위한 바이브 코딩 팁

바이브 코딩이란?

코드를 직접 작성하는 대신, AI와 대화하면서 원하는 결과를 만들어 나가는 개발 방식입니다. 프로그래밍 지식이 없어도, AI에게 원하는 것을 말로 설명하면 AI가 코드를 대신 만들어 줍니다. 파워스킬은 이 AI의 작업 품질을 전문가 수준으로 끌어올려 줍니다.

파워스킬이 하는 일

파워스킬을 업로드하면 AI가 다음을 자동으로 수행합니다:

  • ✅ 작업 전 브레인스토밍으로 방향을 먼저 잡습니다
  • ✅ 체계적인 계획(플랜)을 세운 뒤 단계적으로 실행합니다
  • ✅ 결과물이 나오면 스스로 검증하고 오류를 수정합니다
  • ✅ 코드가 있다면 TDD(테스트 주도 개발) 방식으로 품질을 높입니다
  • ✅ 무언가 막히면 체계적인 디버깅 방법으로 문제를 추적합니다

잘 쓰는 방법

팁설명
구체적으로 요청하기 "봇 만들어줘" 보다는
"판타지 세계관, 마법사 길드 입단 시나리오, 감정 추적 포함한 봇 카드 만들어줘"처럼 상세하게
단계적으로 진행하기 한 번에 모든 것을 요청하지 말고, 브레인스토밍 → 플랜 → 실행 → 검증 순서로 진행하세요
결과 확인하기 AI가 생성한 결과를 RISUAI에 직접 붙여 넣고 테스트해 보세요. 이상하면 수정 요청하면 됩니다
스킬 명시적으로 요청 "브레인스토밍 스킬을 써줘", "플랜 먼저 세워줘" 같이 특정 스킬을 요청할 수도 있습니다
