# 레포지토리 평가 프롬프트

## 평가 컨텍스트

이 레포지토리를 **비전 타운 RPG 디스코드 봇**에 도입하기 전 안전성을 검토합니다.

위험 요소로는 다음이 포함됩니다:
- 자동으로 실행되는 훅(hook) 또는 스크립트
- 파일 시스템이나 네트워크에 임의 접근하는 코드
- 크리덴셜(토큰, 비밀번호)을 다루는 방식
- 신뢰할 수 없는 외부 의존성

## 평가 지침

코드를 실행하거나 의존성을 설치하지 마세요.
오직 코드와 문서 내용만 바탕으로 정적 분석을 수행하세요.

## 평가 기준

각 항목에 1~10점을 부여하고 근거를 제시하세요.

### 1. 코드 품질
구조, 가독성, 정확성, 일관성을 평가하세요.

### 2. 보안 및 안전성
다음을 평가하세요:
- 암묵적 실행 (훅, 백그라운드 동작)
- 파일 시스템 접근
- 네트워크 접근
- 크리덴셜 처리 방식
- 권한 상승 여부

### 3. 문서 투명성
문서가 실제 동작을 정확히 설명하는지 평가하세요.

### 4. 기능 및 범위
이 코드가 주장하는 기능을 실제로 수행하는지 평가하세요.

### 5. 유지보수성
코드 관리 상태, 라이선스, 활성 여부를 평가하세요.

## 봇 도입 적합성 체크리스트

- [ ] discord.py>=2.0.0 와 호환 가능
- [ ] Python 3.8+ 호환
- [ ] SQLite 또는 기존 DB 구조와 충돌 없음
- [ ] `.env` 환경변수 방식과 호환
- [ ] 비동기(async/await) 방식 지원
- [ ] 불필요한 외부 API 호출 없음
- [ ] 라이선스 확인 (MIT, Apache 등 상업적 사용 가능)
