# 코드 최적화 요청 — 비전 타운 RPG 봇

아래 코드의 성능과 품질을 분석하고, 구체적인 개선안을 제안해줘.

## 프로젝트 컨텍스트
- **스택**: Python, discord.py>=2.0.0, SQLite
- **패턴**: 비동기(async/await), @bot.command 데코레이터
- **DB**: SQLite (`get_db_connection()` 사용)
- **제약**: 터미널 없이 Claude Desktop에서만 작업

## 분석 요청 항목

1. **성능 문제**
   - 불필요한 DB 쿼리 중복 여부
   - 무거운 반복문이나 비효율적 로직
   - 비동기 처리 개선 가능한 부분

2. **코드 품질**
   - 중복 코드 제거 가능한 부분
   - 함수 분리 또는 모듈화 제안
   - 에러 핸들링 개선

3. **디스코드 봇 특화**
   - discord.py 2.0+ 모범 사례 적용 여부
   - rate limit 대응 개선
   - 메모리 누수 가능성
