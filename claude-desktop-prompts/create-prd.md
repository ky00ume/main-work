# PRD 작성 프롬프트 — 비전 타운 RPG 봇 신기능

당신은 경험 많은 게임 기획자입니다. 비전 타운 RPG 디스코드 봇에 추가할 새 기능의 PRD(기능 명세서)를 작성해주세요.

## 중요 원칙
- 기술 구현 방법보다 **기능과 유저 경험**에 집중
- 시간 추정치 포함 금지
- 터미널 없이 Claude Desktop만으로 구현 가능한 범위로 작성

## 프로젝트 컨텍스트
- **프로젝트**: 비전 타운 RPG 디스코드 봇
- **스택**: Python + discord.py, SQLite DB
- **기존 시스템**: Player 클래스, BattleEngine, 낚시/요리/제련/채집/퀘스트/NPC/상점 등
- **봇 패턴**: `@bot.command`, `await ctx.send()`, discord.py Embed UI
